## 📖 简介

专为海豹骰设计的 **草群友** 插件，集主动攻击、反击、群P、排行榜、商店、宠物养成于一体，  
让群聊充满 **生草** 与 **鬼畜** 的欢乐氛围。  
**注意**：本插件仅供娱乐，请勿过度认真 😉

---

## 🚀 核心指令

| 指令 | 说明 |
|------|------|
| `.草群友帮助` | 查看完整帮助（所有指令一览） |
| `.草群友 @某人 @骰娘` | **主动** 草对方，获取精华 |
| `.反草 @某人 @骰娘` | **反击** 对方（精华减半，消耗冷却） |
| `.群p @某人 @骰娘` | 参与 **群P接龙**，多人混战 |
| `.精华量排行 [页]` | 查看 **精华排行榜**（分页） |
| `.攻受排行` | 查看 **主动/被动** 排行榜 |
| `.精华查询 @某人 @骰娘` | 查询 **个人档案**（精华数、战绩） |
| `.关系 @某人 @骰娘` | 查看你与对方的 **关系亲密度** |
| `.状态` | 查看自己的 **当前状态**（发情/虚弱等） |
| `.合欢楼` | 打开 **商城** 购买道具 |
| `.购买 商品名 [数量]` | 购买指定道具（可批量） |
| `.背包` | 查看自己拥有的 **道具** |
| `.使用 道具名 [@目标] @骰娘` | 对目标 **使用道具**（效果各异） |
| `.领养触手` | 获得一只 **触手宠物**（初始等级） |
| `.喂养触手` | 喂食触手（消耗 **被动精华** 提升等级） |
| `.触手排行` | 查看全群 **触手等级排行** |
| `.重置精华 @某人 @骰娘` | **[管理员]** 重置指定用户的个人数据 |
| `.清空精华排行` | **[管理员]** 清空全群所有排行数据 |

## 🎭 状态系统

| 状态 | 效果 | 触发条件 |
|------|------|----------|
| **发情** | 主动攻击 × **1.5** 精华 | 每日首次主动时，**10%** 概率触发，持续 **3小时** |
| **虚弱** | 主动攻击 × **0.5** 精华 | 同样概率触发，持续 **3小时** |

> 💡 状态会实时影响你的战斗收益，请留意 `.状态` 查看当前情况。

---

## 🎲 随机事件

每次操作有 **3%** 的概率触发 **特殊事件**，事件内容随机（可能正面也可能负面），  
增加游戏的不可预测性与生草程度 🌿
